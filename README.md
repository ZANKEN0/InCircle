# InnerCircle Frontend

Frontend untuk aplikasi private `InnerCircle Hub`, dibuat dengan Vue 3 dan Vite.

## Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm install
npm run build
```

## Environment

Buat file `.env` dari `.env.example`.

```env
VITE_API_URL=http://localhost:3000/api
```

Jika backend berjalan di domain lain, ganti `VITE_API_URL` sesuai URL API production.
